# Well Paid: Payroll Management System (PMS)

- A LAN-based payroll and attendance system developed for Synergy Grafix Corporation.
- Built with **Java (JDK 23)** and **MySQL 8.0**, featuring secure login and real-time biometric integration.
- Designed to replace the manual Excel-based process with an automated, centralized solution.
- Follows the **Verification and Validation (V-Model)** software development process for structured testing and quality assurance.
- Includes source code and system design assets for educational and practical use.
-(Not for deployment)
